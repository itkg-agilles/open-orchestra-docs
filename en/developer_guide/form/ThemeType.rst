=========
ThemeType
=========


The ThemeType form is used to manage theme flex in Back Office.

+-----------------------------------+--------------------------------------------------+
| **Bundle**                        | OpenOrchestraBackofficeBundle                    |
+-----------------------------------+--------------------------------------------------+
| **Name**                          | oo_theme                                         |
+-----------------------------------+--------------------------------------------------+
| **Class**                         | ThemeType                                        |
|                                   |                                                  |
+-----------------------------------+--------------------------------------------------+
| **Parent type**                   | FormType                                         |
|                                   |                                                  |
+-----------------------------------+--------------------------------------------------+
| **Options**                       |  * inherited options FormType                    |
|                                   |                                                  |
+-----------------------------------+--------------------------------------------------+
| **Fields** (name, type)           | * name       (text)                              |
|                                   |                                                  |
+-----------------------------------+--------------------------------------------------+
